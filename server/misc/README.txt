Descriptions of what all the files in this folder should go here.

* competitor info stuff was designed as a stop gap to
  let competitors see their info.

HOWTO: Update the live site
    1) Run generate* on an instance of the live db.
    2) Save output to CompetitorInfoData.class.php .
    3) Send newly written file to webserver.

* reg reports are a quick hard-coded look at the demographic of
  competitors registered for CMAT16

* badges is a quick way to generate a table suitable for use in
  making badges via Mail Merge or a similar utility.
